📊 主工具｜Performance 數據診斷器
80% 的日常使用都在這一頁：輸入少量數據 → 產生診斷 → 決定下一支影片改什麼

先看這裡｜這個工具怎麼用？

▼
主工具定位：Performance Manual 不是要你從頭讀到尾。日常使用時，先用「數據診斷器」；其他頁面是查表、名詞解釋與進階參考。

Step 1｜選平台與目標

先選 YouTube / Instagram / Facebook，再選這支內容的任務：曝光、信任、教學、私訊或轉化。

Step 2｜填最少 3 個數字

優先填入口數字（CTR / Stop Rate）、留存數字（Retention / APV）、轉化數字（DM / 點擊 / 報名）。沒有的欄位先留空。

Step 3｜看下一步決策

診斷結果會告訴你優先修 Hook、Script、CTA、Trust、Offer，或先檢查 Paid / Organic 是否分開。

其他頁面怎麼用：Meta 2026、三平台指標、四象限、history_tracking 都是「需要時再查」。不要一開始就被全部名詞淹沒。

輸入核心數據｜不知道就留空，不要亂猜

▼
這個表單的用意：不用一次填完整後台所有數字。先填「入口、留存、受眾、轉化」四類核心數據，工具會把數據翻成一句可執行的修改建議：下一支先修 Hook、Script、CTA、Offer，還是受眾定位。
先看這裡｜填寫邏輯：沒有每個數字也可以用。你只要填得出 3 個欄位：①入口數字（CTR 或 Stop Rate）②留存數字（Retention / APV）③轉化數字（DM / 點擊 / 報名），就能先做第一輪診斷。沒有的欄位先留空，不要亂猜。

最小填法 A｜YouTube

填 CTR、Retention/APV、Viewers 或 Views、DM/點擊。CTR 在 Studio → Analytics → Reach。

最小填法 B｜IG / FB Reels

沒有 CTR 沒關係。填 Retention 或平均觀看比例、Viewers/Views、Interactions、DM 或 Profile visits。

最小填法 C｜剛開始沒數據

只填你有的數字。沒有就留空。第一輪目標是建立記錄習慣，不是做精密統計。

欄位怎麼填？去哪裡找？

▼
欄位你要填什麼YouTube 到哪裡找IG / FB 到哪裡找沒有數字時怎麼辦
CTR / Stop Rate（%）YouTube 填 CTR；短影音填「前幾秒停下來的人比例」或用你後台最接近的入口數字。Studio → Analytics → Reach → Impressions CTRReels 通常沒有官方 CTR；可用前 3 秒留存、播放開始率或暫時留空。沒有就留空，先靠 Retention + Viewers + DM 判斷。
Retention / APV（%）平均觀看比例。例：影片 60 秒，平均看 30 秒，約 50%。Studio → Engagement → Audience retention / Average percentage viewedInsights / 專業儀表板裡找平均觀看時間、觀看比例或留存曲線。沒有百分比時，用平均觀看秒數 ÷ 影片長度 × 100 粗估。
Viewers（不重複觀眾）看過這支內容的不同帳號數。YouTube 可先用 Views 近似，但要知道它不是不重複觀眾。Meta 2026 後看 Viewers；舊介面可能叫 Reach / Accounts Reached。只有 Views 時也可先填 Views，但診斷會比較粗。
DM Count / 轉化動作因這支內容帶來的私訊、點擊、報名、購買、索取表單。通常要自己記錄：連結點擊、留言後私訊、表單填寫。DM、Profile visits、Link clicks、私訊關鍵字觸發次數。沒有追蹤就填 0，並在下一支開始記錄。
Paid 是否有投放？這支內容有沒有用廣告推。看是否有 YouTube Ads / Promotion。Meta Business Suite Insights 概覽多為 Organic；Paid 要到 Ads Manager / Page Insights 查。不確定時選「有 Paid，需分開看」，避免誤判。
重要：數據診斷不是越精密越好。第一輪只要能回答：「有沒有人點進來 / 有沒有留下來 / 有沒有走下一步」就夠了。
用意：不同平台數字名稱不同；YouTube 看 CTR / Retention，Meta 2026 後看 Viewers / Views / Interactions。
用意：先知道這支影片要解決什麼任務。曝光片不一定要立刻成交；轉化片則必須看 DM / Offer。
白話：入口吸引力。YouTube CTR 是看到縮圖/標題後點進來的比例；短影音 Stop Rate 是前 1–3 秒有沒有停下來。用意：判斷 Hook、標題、縮圖、前幾秒是否吸引人。
白話：觀眾平均看了影片多少比例。APV = Average Percentage Viewed。用意：判斷腳本節奏、資訊密度、內容是否兌現承諾。
白話：實際看過內容的不同帳號數；同一人看很多次仍算 1。用意：判斷觸及到多少「人」，不是多少次播放。
白話：因這支內容帶來的私訊、點擊、報名、購買等下一步行動。用意：判斷 CTA 和 Offer 是否真的把觀看變成關係或收入。
用意：有投廣告時，不要把買來的曝光誤判成自然內容成功。Paid 與 Organic 必須分開記。
白話：觀眾表示不想看、負面留言、跳出或大量不相關互動。用意：判斷是否受眾錯位或標題承諾過度。

🔄 Meta 2026 指標更新
避免用舊 Reach / Impressions / Engagement 做錯判斷

先看這張｜Meta 新指標白話說明

▼
這張表的用意：先把 Meta 新名詞翻成白話。你不用先懂演算法，只要知道每個欄位回答哪個問題。
新名詞白話意思這欄位用意怎麼看最常見誤讀
Viewers不重複觀眾。同一個人看 5 次，仍算 1 個 viewer。看這支內容實際碰到多少「人」。Viewers 增加，代表受眾面擴大；但仍要搭配留存與互動。不要跟 Views 混用；Viewers 不是播放次數。
Views總播放 / 總觀看次數，可能包含同一人重複觀看。看播放量、重播、擴散熱度。Views 遠高於 Viewers，可能代表有人重看，也可能只是重複曝光。Views 高不等於內容成功，也不等於有轉化。
Interactions主動互動，例如按讚、留言、分享、收藏。看觀眾是否願意做動作。要看互動品質：真實留言、分享、收藏比「+1」有價值。留言數多不等於好；機械式留言可能是風險。
Organic Viewers自然流量帶來的不重複觀眾。看內容本身是否有自然吸引力。評估內容策略時優先看這個。不要把 Paid 帶來的人也算進內容自然成功。
Paid Viewers廣告帶來的不重複觀眾。看投放是否有效。用來評估廣告，不用來判斷自然演算法是否喜歡你。Paid 成功不代表內容自然成功。
如何使用：如果你只想知道「這支內容自然表現好不好」，先看 Organic Viewers + Retention + Interactions。若有廣告，Paid 數字要分開記。

Paid / Organic 正確說法

▼
這段的用意：很多人看到 Business Suite 數字後，以為那就是全部流量。正確做法是先確認你看的介面是不是只含 Organic，若有投廣告，Paid 數字要另外去 Ads Manager / Page Insights 查。

🔁 Data → Diagnosis → Decision → Iteration
Performance Review 的核心不是複盤，而是下一支影片的修改單

Data｜數據

收集 CTR / Stop Rate、Retention、Viewers、Interactions、DM、Revenue。

Diagnosis｜診斷

把數據模式對應到 Hook、Script、CTA、Trust、Offer 或 Audience 問題。

Decision｜決策

決定下一支只修哪一個變數。

Iteration｜迭代

下一支影片只測一件事，不要同時亂改。

診斷 → 決策對照

▼
數據模式優先診斷決策
CTR 低標題 / 縮圖 / 主題包裝重寫入口承諾
Retention 低Script 節奏、資訊密度、承諾未兌現移除 Dips 段落或加快節奏
Views 高但 DM 低CTA / Trust / Offer 問題改 CTA 類型或重設引流品
Paid 高但 Organic 低內容自然吸引力不足分開讀數據，重測 Organic
Not interested 高受眾錯位或包裝誤導重做受眾與 Hook 承諾

  3. 如何看：數字變高或變低時，應該修哪裡。
  4. 可信度：official 是官方定義；high_confidence 是高可信更新；practical_inference 是實務判斷，不可當官方規則。

Performance 常見名詞白話表

▼
名詞白話意思用來判斷什麼不要誤解成
CTR看到標題 / 縮圖後，有多少人點進來。入口包裝：標題、縮圖、題目是否吸引人。CTR 高不等於影片好；還要看 Retention。
Stop Rate短影音前 1–3 秒讓人停下來的能力。Hook、第一句話、第一個畫面是否有效。不是觀看總數。
Retention Rate觀眾平均看了影片多少比例。腳本節奏、資訊密度、內容是否有兌現承諾。不是互動率。
AVDAverage View Duration，平均觀看時長。YouTube 長影片內容是否留得住人。不是總觀看時間。
APVAverage Percentage Viewed，平均觀看比例。不同長度影片之間較好比較的留存比例。不是點擊率。
Viewers不重複觀眾數，同一帳號多次觀看仍算 1。看實際觸及多少人。不是播放次數。
Views總觀看 / 播放次數，可能含重播。看播放量、重看、擴散量。Views 高不等於轉化好。
Interactions主動互動，如按讚、留言、分享、收藏。看內容是否讓人願意行動。留言多不等於互動好，要看是否真實。
DM Count私訊數或明確轉化動作數。看 CTA / Offer 是否帶人進下一步。不是官方推播權重。
Organic自然流量，沒有靠廣告買來。評估內容本身是否有效。不要跟 Paid 混著判斷。
Paid廣告投放帶來的流量。評估廣告效益。不要當成自然演算法成功。
Not Interested觀眾表示不想看或負向回饋。判斷受眾錯位、標題過度承諾、內容不匹配。不是一般負評而已，要優先檢查。

入口低

CTR / Stop Rate 低 → 先修 Hook、標題、縮圖、第一句。

留不住

有看沒行動

Retention 高但 DM / Conversion 低 → 修 CTA、Trust、Offer。

數字怪

先檢查 Paid / Organic 是否分開，再下結論。
